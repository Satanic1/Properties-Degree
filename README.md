# Properties-Degree
Понятие степени числа является одним из фундаментальных в математической науке. Оно формировалось на протяжении тысячелетий: от древнегреческих геометрических представлений Евклида до трудов Рене Декарта, утвердившего современную запись, и Исаака Ньютона, расширившего понятие на дробные и иррациональные показатели .
